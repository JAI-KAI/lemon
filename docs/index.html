關於檸檬先生

檸檬先生創立於2001年，從小小的農產行到現在發展成「哲昌農產有限公司」。 現代人對於養身天然的食品越來越講究，本公司秉持著天然水果/原汁營業至今， 是一間位於台中、有註冊商標且符合政府相關法令與標準的公司。 為了穩定貨源我們在屏東縣政府-鹽埔鄉農會成立農業產銷第29班，除了在屏東租用農地11甲自產自銷以外， 也和在地農民培養信任關係、定期收購水果，積極推銷在地水果。另外，在台中青果菜市場也經營攤位以供不同客層的消費者購買。
